The scrape API extracts structured data from specific elements on a webpage using CSS selectors, returning details such as element dimensions and inner HTML for efficient data extraction.

## Basic usage

```bash
curl -X POST 'https://api.cloudflare.com/client/v4/accounts/<accountId>/browser-rendering/scrape' \
  -H 'Authorization: Bearer <apiToken>' \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "https://example.com/",
    "scrapeOptions": {
      "selectors": ["h1", ".article"]
    }
  }'
```

### JSON response

```json title="json response"
{
	"success": true,
	"result": [
		{
			"results": [

					"attributes": [],

					"text": "Example Domain",
					"top": 133.4375,
					"width": 600
				}
			],
			"selector": "h1"
		},
		{
			"results": [
				{
					"attributes": [

	]
}
```
